## Description

[Nest](https://github.com/ocamilomontealegre/user-microservice) User microservice for the social network app

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
